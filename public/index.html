RESTful API For Events

Add event:

Endpoint to add a new event to the database

CRUD: POST
Endpoint: /events


Retrieve all events:

Endpoint to retrieve all event data from the database

CRUD: GET
Endpoint: /events


Retrieve event by ID:

Endpoint to retrieve a given event by ID

CRUD: GET
Endpoint: /events/{id}


Retrieve event by name:

Endpoint to retrieve a given event by name

CRUD: GET
Endpoint: /events/by-name/{name}


Update event by id:

Endpoint to update a given event by ID

CRUD: PUT
Endpoint: /events/{id}


Delete event by id:

Endpoint to delete a given event by ID

CRUD: DELETE
Endpoint: /events/{id}


Delete all events:

Endpoint to delete all events from the database

CRUD: DELETE
Endpoint: /events
